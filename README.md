# FinDash — Интерактивный аналитический дашборд

**[Смотреть Live Demo](https://8wuluwulu.github.io/dashboard-project/)** FinDash — это статический дашборд для финансовой аналитики, построенный по принципам Single-Page Application (SPA) без использования тяжелых фреймворков. Проект демонстрирует навыки построения сложных интерфейсов, работы с DOM и интеграции сторонних библиотек для визуализации данных.

## Ключевые особенности

* **SPA Routing (Hash-based):** Мгновенное переключение между вкладками (Обзор, Аналитика, Транзакции) без перезагрузки страницы, с полной поддержкой истории браузера (кнопки "Назад/Вперед").
* **Интерактивные графики:** Интеграция библиотеки **Chart.js** для визуализации динамики выручки и источников трафика.
* **Сложная CSS-архитектура:** Построение адаптивной сетки на базе `CSS Grid` и `Flexbox`. Использование CSS-переменных (`:root`) для управления премиальной темной темой.
* **Адаптивность (Mobile-First):** Корректное отображение на любых устройствах.
* **Vanilla JavaScript:** Вся логика роутинга, модальных окон и рендеринга данных написана на чистом JS.

## Стек технологий

* **Разметка & Стили:** HTML5, CSS3 (Grid, Flexbox, Variables)
* **Логика:** Vanilla JavaScript (ES6+)
* **Библиотеки:** Chart.js (CDN)
* **Инструменты:** Git, GitHub Pages, AI-Assisted Development

## AI-Assisted Workflow

Сложная логика (кастомный Hash-роутер с паттерном "Silent Default", интеграция Chart.js, исправление багов истории браузера) была спроектирована и реализована вручную. Это позволило сократить время на написание шаблонного кода и сфокусироваться на архитектуре и пользовательском опыте.
